# 🛰️ Terra Classic: Reconstruction Map (v1.0)

A real-time, 3D geospatial and data-driven visualization of the **Terra Classic (LUNC)** blockchain. This project monitors the network's recovery through visual metaphors

## 🛠️ Tech Stack
- **Engine:** Three.js (Vanilla JavaScript)
- **Data:** Terra Classic Public LCD / WebSocket
- **Hosting:** GitHub Pages (Free)
